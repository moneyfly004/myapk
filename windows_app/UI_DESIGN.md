## 详细界面元素

### 1. 标题栏（顶部）
```
┌─────────────────────────────────────────────┐
│ 🔑 NEKOBOX                    ☰            │
└─────────────────────────────────────────────┘
```
- **背景**: 深黑色半透明 (#050508, 80%)
- **底部边框**: 青色霓虹线（1px，30% 不透明度）
- **左侧图标**: 🔑 VPN 钥匙图标（青色，28px）
- **标题文字**: "NEKOBOX"（青色霓虹，22px，粗体，字间距 3px）
- **右侧按钮**: ☰ 菜单按钮（青色霓虹）

### 2. 用户信息卡片
```
┌─────────────────────────────────────┐
│ 到期时间：未设置                      │ ← 青色霓虹文字
│ ████████████░░░░░░░░░░░░░░░░░░░░░  │ ← 青色进度条
│ 剩余天数：0                           │ ← 灰色小字
│ 设备数：0/0           在线：0       │ ← 左右分布
└─────────────────────────────────────┘
```
- **边框**: 青色霓虹（1.5px，50% 不透明度）
- **背景**: 深色半透明 + 毛玻璃效果
- **发光**: 青色阴影（20px 模糊，30% 不透明度）
- **圆角**: 16px

### 3. 连接按钮（居中，最突出）
```
        ┌──────────────┐
        │              │
        │  开始连接     │ ← 大型按钮（240x70）
        │ (发光脉冲)    │   青色霓虹发光
        │              │   连接时：脉冲动画
        └──────────────┘   断开时：金色霓虹
```
- **尺寸**: 240px × 70px（大型，居中）
- **边框**: 青色霓虹（2px）
- **文字**: "开始连接" 或 "断开连接"（青色/金色，20px，粗体）
- **动画**: 
  - 连接时：持续发光脉冲（0.3-1.0 不透明度循环）
  - 断开时：金色霓虹 (#FFD700)
- **阴影**: 多层青色发光阴影（20px + 40px 模糊）

### 4. 路由模式选择器
```
┌─────────────────────────────────────┐
│ 路由模式                             │ ← 粉色霓虹标题
│ ┌──────────────┐ ┌──────────────┐  │
│ │ 规则模式     │ │ 全局模式     │  │ ← 两个选项
│ │ (选中-发光)  │ │              │  │   选中：发光边框
│ └──────────────┘ └──────────────┘  │
│ 规则模式：规则内不翻墙，规则外翻墙   │ ← 灰色提示
└─────────────────────────────────────┘
```
- **边框**: 粉色霓虹 (#FF00FF)
- **选项**:
  - 规则模式：青色霓虹（选中时发光）
  - 全局模式：绿色霓虹（选中时发光）
- **选中效果**: 边框加粗、背景高亮、发光阴影

### 5. 节点选择器（可展开）
```
┌─────────────────────────────────────┐
│ 节点选择    自动选择 (Auto) ▼      │ ← 标题栏（可点击）
├─────────────────────────────────────┤
│ ☁️  自动选择 (Auto)   延迟: 50ms ✓ │ ← 选中项（绿色发光）
│ ☁️  节点 1 - 香港     延迟: 50ms   │
│ ☁️  节点 2 - 日本     延迟: 80ms   │
│ ☁️  节点 3 - 美国     延迟: 120ms  │
└─────────────────────────────────────┘
```
- **边框**: 青色霓虹
- **节点项**:
  - ☁️ 云图标（青色，24px）
  - 节点名称（14px）
  - 延迟信息（灰色，12px）
  - 选中标记（✓，绿色，24px）
- **选中效果**: 边框加粗、背景高亮、发光阴影

## 视觉效果细节

### 赛博朋克特色效果

1. **霓虹发光**:
   ```
   边框: 青色霓虹线（1.5-2px）
   阴影: 多层阴影（20px + 40px 模糊）
   文字: 发光阴影（10px + 20px + 30px）
   ```

2. **网格背景**:
   ```
   颜色: 青色 (#00FFFF)
   间距: 20px
   透明度: 10%
   覆盖: 整个背景
   ```

3. **毛玻璃效果**:
   ```
   模糊: 10px (BackdropFilter)
   背景: 半透明深色
   质感: 磨砂玻璃
   ```

4. **渐变背景**:
   ```
   从: #0A0A0F (深黑)
   到: #050508 (更深黑)
   方向: 顶部到底部
   ```

### 动画效果

1. **连接按钮脉冲**:
   - 不透明度: 0.3 → 1.0 → 0.3（循环）
   - 持续时间: 1.5秒
   - 缓动: easeInOut

2. **节点展开**:
   - 平滑展开/收起
   - 高度动画

3. **选中状态**:
   - 边框发光增强
   - 背景高亮
   - 即时反馈

## 颜色方案

### 霓虹色彩
- **主色（青色）**: #00FFFF - 主要元素、边框、文字
- **次色（粉色）**: #FF00FF - 辅助元素、路由选择器
- **强调（绿色）**: #00FF00 - 选中状态、在线状态
- **警告（金色）**: #FFD700 - 断开状态

### 背景色
- **主背景**: #0A0A0F（深黑色）
- **深色背景**: #050508（更深黑）
- **卡片背景**: #1A1A2E（80% 不透明度）
- **表面背景**: #16213E

### 文字颜色
- **主文字**: #E0E0E0（浅灰色）
- **次文字**: #B0B0B0（中灰色）
- **霓虹文字**: #00FFFF（青色霓虹）

## 布局特点

### 垂直布局
1. **标题栏**（固定顶部，60px）
2. **用户信息卡片**（顶部区域）
3. **连接按钮**（居中，最突出）
4. **路由模式选择器**（中间区域）
5. **节点选择器**（底部区域）

### 间距设计
- **大间距**: 32px（主要元素之间）
- **中间距**: 16px（卡片内部）
- **小间距**: 8px（紧密元素）

### 对齐方式
- **水平**: 居中对齐（连接按钮）
- **垂直**: 顶部对齐（标题栏）
- **内容**: 左右分布（设备信息）

## 交互反馈

### 按钮状态
- **正常**: 青色霓虹，50% 不透明度
- **悬停**: 发光增强，70% 不透明度
- **点击**: 短暂闪烁
- **连接中**: 持续脉冲动画
- **已连接**: 金色霓虹
- **禁用**: 降低不透明度

### 卡片状态
- **正常**: 标准发光
- **选中**: 发光增强 + 背景高亮
- **点击**: 轻微缩放

### 列表状态
- **正常项**: 标准边框
- **选中项**: 加粗边框 + 发光 + 背景高亮
- **悬停**: 边框发光增强
