# 🎨 Windows 桌面应用界面预览

## 整体设计风格

### 赛博朋克风格主题
- **背景色**: 深黑色 (#0A0A0F) 渐变
- **网格背景**: 青色网格线（透明度 10%）
- **主色调**: 青色霓虹 (#00FFFF)
- **辅助色**: 粉色霓虹 (#FF00FF)、绿色霓虹 (#00FF00)
- **文字颜色**: 浅灰色 (#E0E0E0)

## 详细界面元素

### 1. 顶部标题栏
- **高度**: 60px
- **背景**: 深黑色半透明 (#050508, 80% 不透明度)
- **底部边框**: 青色霓虹线（1px，30% 不透明度）
- **左侧**: 
  - 🔑 图标（青色，28px）
  - "NEKOBOX" 文字（青色霓虹，22px，字间距 3px）
- **右侧**: 
  - ☰ 菜单按钮（青色霓虹）

### 2. 用户信息卡片
- **样式**: 霓虹边框卡片（青色边框，1.5px）
- **背景**: 深色半透明 (#1A1A2E, 80% 不透明度)
- **毛玻璃效果**: BackdropFilter 模糊
- **发光效果**: 青色阴影（20px 模糊，30% 不透明度）
- **内容**:
  - 到期时间文字（青色霓虹，16px，粗体）
  - 进度条（青色，8px 高）
  - 剩余天数（灰色，12px）
  - 设备数和在线数（14px，左右分布）

### 3. 连接按钮（居中）
- **尺寸**: 240px × 70px
- **样式**: 大型霓虹按钮
- **边框**: 青色霓虹（2px）
- **背景**: 半透明渐变
- **文字**: "开始连接" 或 "断开连接"（青色霓虹，20px，粗体，字间距 2px）
- **动画**: 
  - 连接时：发光脉冲动画（0.3-1.0 不透明度循环）
  - 断开时：金色霓虹 (#FFD700)
- **阴影**: 多层青色发光阴影

### 4. 路由模式选择器
- **卡片样式**: 粉色霓虹边框
- **标题**: "路由模式"（粉色霓虹，18px，粗体，字间距 2px）
- **选项**:
  - 规则模式（左侧，青色霓虹）
  - 全局模式（右侧，绿色霓虹）
- **选中效果**: 
  - 边框加粗（2px）
  - 背景高亮（10% 不透明度）
  - 发光阴影
- **提示文字**: 灰色，12px

### 5. 节点选择器
- **卡片样式**: 青色霓虹边框
- **标题栏**（可点击展开）:
  - "节点选择"（左侧，青色霓虹，18px，粗体）
  - 当前选中节点（右侧，绿色霓虹，14px）
  - 展开/收起箭头（青色）
- **节点列表**（展开时）:
  - 每个节点项：
    - ☁️ 云图标（青色，24px）
    - 节点名称（14px）
    - 延迟信息（灰色，12px）
    - 选中标记（✓，绿色，24px）
  - 选中效果：
    - 边框加粗（2px）
    - 背景高亮
    - 发光阴影

## 视觉效果

### 赛博朋克特色
1. **霓虹发光**:
   - 所有边框都有发光效果
   - 文字有阴影发光
   - 按钮有脉冲动画

2. **网格背景**:
   - 青色网格线（20px 间距）
   - 10% 不透明度
   - 覆盖整个背景

3. **毛玻璃效果**:
   - 卡片使用 BackdropFilter
   - 10px 模糊半径
   - 半透明背景

4. **渐变背景**:
   - 深黑色渐变
   - 从顶部到底部
   - 营造深度感

### 动画效果
1. **连接按钮**:
   - 连接时：持续发光脉冲
   - 断开时：静态金色

2. **节点展开**:
   - 平滑展开/收起动画

3. **选中状态**:
   - 边框发光增强
   - 背景高亮
