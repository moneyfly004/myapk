## 🎯 操作步骤对比

### 连接 VPN

#### 原版流程
```
1. 打开应用
2. 在配置列表中找到节点
3. 点击节点（或点击节点的菜单）
4. 选择"连接"
5. 点击底部浮动按钮确认
6. VPN 连接
```
**需要 5 步操作！**

#### 修改版流程
```
1. 打开应用
2. 点击超大"连接"按钮
3. VPN 连接
```
**只需 2 步！** 🎉

---

### 切换节点

#### 原版流程
```
1. 断开当前连接
2. 在配置列表滚动查找节点
3. 点击新节点
4. 点击连接按钮
5. VPN 重新连接
```
**需要 5 步操作！**

#### 修改版流程
```
1. 点击"当前节点"卡片
2. 节点列表弹出（自动测速排序）
3. 点击要切换的节点
4. 自动切换完成
```
**只需 3 步，且自动切换！** 🎉

---

### 切换模式

#### 原版流程
```
1. 打开侧边菜单
2. 点击"设置"
3. 找到"代理模式"选项
4. 选择模式
5. 返回主页
6. 重新连接生效
```
**需要 6 步操作！**

#### 修改版流程
```
1. 点击模式按钮（规则/全局）
2. 立即生效
```
**只需 1 步！** 🎉
